# Measure Deck

A Blackwellen Ltd product. Repository: https://github.com/Blackwellen/measure-deck

## What it is
Measure Deck is a web application built by Blackwellen Ltd. The repository
README currently describes it only as "a Next.js 16 application for
MeasureDeck"; a fuller public product description is not yet published.

## Tech stack
Next.js 16, TypeScript, Supabase (PostgreSQL). The repository's configuration
references Stripe (tiered Starter / Professional / Enterprise plans), an AI
provider integration, Resend for email and Cloudflare R2 for file storage.
Vercel-ready deployment.
